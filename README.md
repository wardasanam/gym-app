# **ForgeFit: Beyond the Limit**

ForgeFit is a high-end, premium fitness application designed with a **Kinetic Editorial** aesthetic. It moves beyond standard gym management into the realm of biological engineering and athletic architecture. This platform serves as the digital front-end for a high-performance training facility, featuring interactive dossiers, technical metrics, and architectural physical recalibration protocols.

To run this project locally, follow these steps:

1. **Clone the repository:**  
   git clone \[https://github.com/wardasanam/gym-app.git\](https://github.com/wardasanam/gym-app.git)  
   cd gym-app

2. **Install dependencies:**  
   npm install

3. **Start the development server:**  
   npm run dev

4. **Build for production:**  
   npm run build
